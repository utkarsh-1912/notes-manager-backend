Notes Manager API Documentation

Welcome to the Notes Manager API! Below is a list of all the available routes and their descriptions. Use this as a guide to interact with the API effectively.

Base URL

Local: http://localhost:5000/api
Production: https://notes-manager-backend-a6z3.onrender.com/api

Routes

GET /notes

Fetch all notes or filter them based on search query and category.

  • Parameters:
    • search: (optional) Search query for filtering notes.
    • category: (optional) Filter notes by category.
  • Response: List of notes.
Example: GET /notes?search=hello&category=others

POST /notes

Create a new note.

  • Body:
    • title: (string) The title of the note.
    • description: (string) The content of the note.
    • category: (string) The category of the note.
  • Response: The created note object.
Example: POST /notes
Body: { "title": "New Note", "description": "Content", "category": "Work" }

GET /notes/:id

Fetch a single note by its ID.

  • Response: Note object.
Example: GET /notes/6741861a96c4cd37a7ddb7b7

PUT /notes/:id

Update an existing note by its ID.

  • Body:
    • title: (string, optional) The updated title.
    • description: (string, optional) The updated content.
    • category: (string, optional) The updated category.
  • Response: The updated note object.
Example: PUT /notes/6741861a96c4cd37a7ddb7b7
Body: { "title": "Updated Note", "description": "Updated Content" }

DELETE /notes/:id

Delete a note by its ID.

  • Response: Success message or status.
Example: DELETE /notes/6741861a96c4cd37a7ddb7b7

Setup Instructions

Follow these steps to set up the project:

  1. Clone the repository.
  2. Install dependencies: npm install
  3. Set environment variables for MONGO_URI and PORT.
  4. Start the server: npm start

This application is built using Express.js for the backend and React for the frontend. For further details, check the project repository.
